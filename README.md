# tms 系统

> 采用 `Vue3` + `Vite2` + `Element Plus` 编写

## 安装使用

- 获取项目代码

```bash
https: git clone -b next https://gitee.com/ningbo-discovery-international/web-tms-system.git
ssh: git clone -b next git@gitee.com:ningbo-discovery-international/web-tms-system.git
```

- 安装依赖

```bash
cd web-tms-system

pnpm install // yarn or npm
```

## 提交规范

- 参考 [vue](https://github.com/vuejs/vue/blob/dev/.github/COMMIT_CONVENTION.md) 规范 ([Angular](https://github.com/conventional-changelog/conventional-changelog/tree/master/packages/conventional-changelog-angular))

  - `feat` 增加新功能
  - `fix` 修复问题/BUG
  - `style` 代码风格相关无影响运行结果的
  - `perf` 优化/性能提升
  - `refactor` 重构
  - `revert` 撤销修改
  - `test` 测试相关
  - `docs` 文档/注释
  - `chore` 依赖更新/脚手架配置修改等
  - `workflow` 工作流改进
  - `ci` 持续集成
  - `types` 类型定义文件更改
  - `wip` 开发中
